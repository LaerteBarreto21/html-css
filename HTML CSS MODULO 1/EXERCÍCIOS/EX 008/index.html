Principais Formatações

• Negrito/Destaque

Não Semantica:

Nessa frase temos este Termo em Negrito
usando a tag <b> e </b>

Semantica:(Significado)

Nesta frase tenho um Termo em Destaque
usando a tag <strong> e </strong>

• Italico

Não Semantica

Nesta frase temos um termo em Italico
usando a tag <i> e < /i >

Semantica:(Significado)

Nesta frase temos um termo em Itálico
usando a tag <em> e </em>

• Marcador de Texto

Podemos criar um Texto marcado
usando a tag <MARK> e </MARK>

• Texto Grande e Pequeno

Estamos Criando um Texto grande e um texto pequeno
usando a tag <big ou small> e </big ou small>
OBS: big não deve mais ser Usado.

• Texto Deletado

Podemos maracar um texto como excluido para indicar que ele deve ser lido mas não considerado.
usando a tag <del> e </del>

• Texto Inserido

Podemos marcar um texto como inserido para dar uma enfase e indicar que ele foi adicionado depois.
usando a tag <ins> e </ins>
existe tambem o sublinhado com a tag u

• Sobrescrito

Para Inserir coisas como X20+2
Usando a tag <sup> e </sup>.

• Subscrito

Para Inserir coisas como H2O
Usando a tag <sub> e </sub>.
